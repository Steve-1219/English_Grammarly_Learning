# 第一章 · 名词、代词

## 一名词（Noun）

> **名词**是用来表示人、事物、地点、抽象概念等名称的词。

### 1. 名词的分类

|      类别      | 说明                                   | 示例                               |
| :------------: | -------------------------------------- | ---------------------------------- |
|  **可数名词**  | 可以用数目计算，有单复数形式           | a book, two apples, three students |
| **不可数名词** | 无法用数目计算，没有复数形式           | water, music, information, money   |
|  **专有名词**  | 特定的人名、地名、机构名等，首字母大写 | China, Tom, London, Microsoft      |
|  **普通名词**  | 一类人或事物的通用名称                 | teacher, city, dog, table          |

### 2. 可数名词复数变化规则

| 规则                                      | 示例                                      |
| :---------------------------------------- | ----------------------------------------- |
| 一般情况加 **-s**                         | book → books, cat → cats                  |
| 以 s, x, sh, ch 结尾加 **-es**            | bus → buses, box → boxes, watch → watches |
| 以辅音字母 + y 结尾，变 y 为 i 加 **-es** | city → cities, baby → babies              |
| 以 f / fe 结尾，变 f / fe 为 v 加 **-es** | knife → knives, leaf → leaves             |
| 不规则变化（需单独记忆）                  | man → men, child → children, foot → feet  |

### 3. 不可数名词的量化表达

不可数名词不能直接用数字修饰，需要借助量词：

- a glass of **water**
- a piece of **information**
- a cup of **tea**
- a bottle of **milk**


## 二代词（Pronoun）

> **代词**是用来代替名词或名词短语的词，避免重复。

### 1. 人称代词

|     人称     | 主格（作主语） | 宾格（作宾语） |
| :----------: | :------------: | :------------: |
| 第一人称单数 |       I        |       me       |
| 第二人称单数 |      you       |      you       |
| 第三人称单数 | he / she / it  | him / her / it |
| 第一人称复数 |       we       |       us       |
| 第二人称复数 |      you       |      you       |
| 第三人称复数 |      they      |      them      |

**例句：**
- **I** love English. （主格作主语）
- Please tell **me** the truth. （宾格作宾语）

### 2. 物主代词

|     人称     | 形容词性（+ 名词） | 名词性（单独使用） |
| :----------: | :----------------: | :----------------: |
| 第一人称单数 |         my         |        mine        |
| 第二人称单数 |        your        |       yours        |
| 第三人称单数 |  his / her / its   |  his / hers / its  |
| 第一人称复数 |        our         |        ours        |
| 第二人称复数 |        your        |       yours        |
| 第三人称复数 |       their        |       theirs       |

**例句：**
- This is **my** book. （形容词性，修饰名词）
- This book is **mine**. （名词性，单独使用）

### 3. 反身代词

表示“自己”，用于强调或表示动作回到主语自身。

|     人称     |          反身代词          |
| :----------: | :------------------------: |
| 第一人称单数 |           myself           |
| 第二人称单数 |          yourself          |
| 第三人称单数 | himself / herself / itself |
| 第一人称复数 |         ourselves          |
| 第二人称复数 |         yourselves         |
| 第三人称复数 |         themselves         |

**例句：**
- I taught **myself** English. （我自学了英语）
- She hurt **herself**. （她伤到了自己）

### 4. 指示代词

|   代词    | 用法               | 示例              |
| :-------: | ------------------ | ----------------- |
| **this**  | 指代近处的单数事物 | This is my pen.   |
| **that**  | 指代远处的单数事物 | That is your bag. |
| **these** | 指代近处的复数事物 | These are apples. |
| **those** | 指代远处的复数事物 | Those are books.  |


## ⚠️ 常见易错点

1. **人称代词主格和宾格混淆**
   - ❌ Me am a student.
   - ✅ **I** am a student.

2. **物主代词形容词性和名词性混淆**
   - ❌ This book is my.
   - ✅ This book is **mine**.

3. **反身代词误用**
   - ❌ I taught me English.
   - ✅ I taught **myself** English.

4. **可数/不可数名词误用**
   - ❌ I need some informations.
   - ✅ I need some **information**.


## 📌 小结

| 词类 | 核心功能           | 关键考点                      |
| :--: | ------------------ | ----------------------------- |
| 名词 | 表示人、事物、概念 | 可数/不可数，单复数变化       |
| 代词 | 代替名词           | 主格/宾格，物主代词，反身代词 |
