# 第一章 · 名词和代词

> 📚 **名词**是英语的基石，**代词**是英语的枢纽。掌握好它们，就搭起了英语表达的基本框架。


## 第一部分 · 名词（Noun）

> **名词**是用来表示人、事物、地点、抽象概念等名称的词，是句子中最基本的成分之一。


### 1. 名词的分类

|                类别                | 说明                                   | 示例                                                         |
| :--------------------------------: | -------------------------------------- | ------------------------------------------------------------ |
|   **可数名词（Countable Noun）**   | 可以用数目计算，有单复数形式           | a book（一本书）, two apples（两个苹果）, three students（三个学生） |
| **不可数名词（Uncountable Noun）** | 无法用数目计算，没有复数形式           | water（水）, music（音乐）, information（信息）, money（钱） |
|    **专有名词（Proper Noun）**     | 特定的人名、地名、机构名等，首字母大写 | China（中国）, Tom（汤姆）, London（伦敦）, Microsoft（微软公司） |
|    **普通名词（Common Noun）**     | 一类人或事物的通用名称                 | teacher（老师）, city（城市）, dog（狗）, table（桌子）      |

#### 普通名词的细分

普通名词根据含义进一步分为四类：

|              类别               | 说明                           | 示例                                                         |
| :-----------------------------: | ------------------------------ | ------------------------------------------------------------ |
| **个体名词（Individual Noun）** | 表示某类人、事物中的个体       | teacher（老师）, student（学生）, dog（狗）, book（书）, house（房子） |
| **集体名词（Collective Noun）** | 表示由若干个体组成的集合体     | family（家庭）, team（团队）, class（班级）, audience（观众）, police（警察） |
|  **物质名词（Material Noun）**  | 表示无法分为个体的物质或材料   | water（水）, air（空气）, wood（木材）, paper（纸）, iron（铁） |
|  **抽象名词（Abstract Noun）**  | 表示动作、状态、品质等抽象概念 | love（爱）, happiness（幸福）, education（教育）, honesty（诚实） |

> 💡 **关键理解**：
> - **个体名词**通常可数：`a teacher`（一位老师）、`three dogs`（三只狗）
> - **集体名词**可单可复：`My family is large.`（我的家庭很大。——视为整体，用单数）/ `My family are all tall.`（我的家人都很高。——视为家庭成员，用复数）
> - **物质名词**和**抽象名词**通常不可数：`some water`（一些水）、`great love`（伟大的爱）


### 2. 可数名词的复数变化

#### （1）规则变化

| 规则                                                       | 示例                                                         |
| :--------------------------------------------------------- | ------------------------------------------------------------ |
| 一般情况加 **-s**                                          | book（书）→ books, cat（猫）→ cats                           |
| 以 s, x, sh, ch 结尾加 **-es**                             | bus（公交车）→ buses, box（盒子）→ boxes, watch（手表）→ watches |
| 以辅音字母 + y 结尾，变 y 为 i 加 **-es**                  | city（城市）→ cities, baby（婴儿）→ babies                   |
| 以元音字母 + y 结尾，直接加 **-s**                         | boy（男孩）→ boys, day（日子）→ days, key（钥匙）→ keys      |
| 以 f / fe 结尾，变 f / fe 为 v 加 **-es**                  | knife（小刀）→ knives, leaf（树叶）→ leaves                  |
| 以 f / fe 结尾，直接加 **-s**（特例）                      | roof（屋顶）→ roofs, chief（首领）→ chiefs, belief（信念）→ beliefs |
| 以辅音字母 + o 结尾，一般加 **-es**（📌 英雄爱吃土豆番茄🍅） | hero（英雄）→ heroes, potato（土豆）→ potatoes, tomato（番茄）→ tomatoes |
| 以元音字母 + o 结尾，直接加 **-s**                         | radio（收音机）→ radios, zoo（动物园）→ zoos, studio（录音室）→ studios |
| 以 o 结尾的外来词/缩写词，直接加 **-s**                    | piano（钢琴）→ pianos, photo（照片）→ photos, kilo（公斤）→ kilos |

> 📌 **补充提示**：
> - 元音字母（a, e, i, o, u）+ y → 直接加 **-s**（如：boy → boys）
> - f/fe 结尾：多数变 f/fe 为 v 加 -es，也有直接加 -s 的特例（如：roof → roofs），需单独记忆
> - o 结尾：辅音 + o 一般加 -es（如：hero → heroes），元音 + o 及外来词/缩写词直接加 -s

#### （2）不规则变化

| 变化类型             |                             单数                             |               复数 |
| :------------------- | :----------------------------------------------------------: | -----------------: |
| 变内部元音           |                         man（男人）                          |      men（男人们） |
| 变内部元音           |                        woman（女人）                         |    women（女人们） |
| 变内部元音           |                          foot（脚）                          |       feet（双脚） |
| 变内部元音           |                        tooth（牙齿）                         |    teeth（牙齿们） |
| 变内部元音           |                        mouse（老鼠）                         |     mice（老鼠们） |
| 变内部元音           |                         goose（鹅）                          |      geese（鹅群） |
| 词尾加 -en / -ren    |                        child（孩子）                         | children（孩子们） |
| 词尾加 -en / -ren    |                          ox（公牛）                          |     oxen（公牛群） |
| 单复数同形           | sheep（羊）, deer（鹿）, fish（鱼）, species（物种）, series（系列）, means（手段）, aircraft（飞行器） |               同左 |
| 外来词（保留原复数） | criterion（标准）→ criteria（多个标准）, phenomenon（现象）→ phenomena（多种现象）, bacterium（细菌）→ bacteria（细菌们）, analysis（分析）→ analyses（多次分析）, thesis（论文）→ theses（多篇论文）, crisis（危机）→ crises（多次危机） |                    |
| 复合名词变复数       | passer-by（路人）→ passers-by（路人们）, mother-in-law（岳母）→ mothers-in-law（多位岳母）, grown-up（成年人）→ grown-ups（成年人们）, looker-on（旁观者）→ lookers-on（旁观者们） |                    |
| 复合名词整体变复数   | toothbrush（牙刷）→ toothbrushes（牙刷们）, bookcase（书柜）→ bookcases（书柜们）, bedroom（卧室）→ bedrooms（卧室们） |                    |


### 3. 某国人的英文表达

|          类型          | 规律            | 示例                                                         |
| :--------------------: | --------------- | ------------------------------------------------------------ |
| **A类**（单复数同形）  | 无需变化        | Chinese（中国人/多个中国人）, Japanese（日本人/多个日本人）, Swiss（瑞士人/多个瑞士人）, Portuguese（葡萄牙人/多个葡萄牙人） |
| **B类**（-man → -men） | 变 -man 为 -men | Englishman（英国男人）→ Englishmen（英国男人们）, Frenchman（法国男人）→ Frenchmen（法国男人们）, Dutchman（荷兰男人）→ Dutchmen（荷兰男人们） |
|  **C类**（规则加 -s）  | 直接加 -s       | German（德国人）→ **Germans**（德国人们）, Russian（俄罗斯人）→ Russians（俄罗斯人们）, Italian（意大利人）→ Italians（意大利人们）, American（美国人）→ Americans（美国人们）, Korean（韩国人）→ Koreans（韩国人们）, Indian（印度人）→ Indians（印度人们） |

> ⚠️ **特别注意**：**German（德国人）的复数是 Germans**，不是 Germen！

**巧记口诀：**
> **中日瑞葡永不变，英法荷兰变 man，其他各国加 -s，德国一定别记错！**


### 4. 名词的所有格

表示所属关系，相当于汉语的"……的"。

#### （1）-'s 所有格（主要用于有生命的名词）

|          情况          | 规则                       | 示例                                                  |
| :--------------------: | -------------------------- | ----------------------------------------------------- |
|        单数名词        | 词尾加 **-'s**             | Tom's book（汤姆的书）, the dog's tail（狗的尾巴）    |
|  以 -s 结尾的复数名词  | 词尾加 **-'**（撇号）      | the students' books（学生们的书）                     |
| 不以 -s 结尾的复数名词 | 词尾加 **-'s**             | children's toys（孩子们的玩具）, men's room（男厕所） |
|        复合名词        | 在最后词尾加 **-'s**       | my mother-in-law's house（我岳母的房子）              |
|    并列名词共同所有    | 只在最后一个名词加 **-'s** | Tom and Jerry's room（汤姆和杰瑞共用的房间）          |
|    并列名词各自所有    | 分别加 **-'s**             | Tom's and Jerry's rooms（汤姆和杰瑞各自的房间）       |

#### （2）of 所有格（主要用于无生命的名词）

| 用法                     | 示例                                                       |
| :----------------------- | ---------------------------------------------------------- |
| 表示无生命事物的所属关系 | the door **of** the room（房间的门）                       |
| 表示较长的定语           | the name **of** the student（学生的名字）                  |
| 表示数量/度量            | a cup **of** tea（一杯茶）, a piece **of** paper（一张纸） |

#### （3）双重所有格（of + 's）

表示"其中一个"或"部分"的概念：

- a friend **of my father's**（我爸爸的一位朋友）
- a book **of Tom's**（汤姆的一本书）
- some students **of mine**（我的一些学生）

#### （4）-'s 所有格的特殊用法

| 用法          | 示例                                                         |
| :------------ | ------------------------------------------------------------ |
| 表示时间      | today's newspaper（今天的报纸）, an hour's drive（一小时的车程） |
| 表示距离      | a mile's walk（一英里的步行路程）                            |
| 表示国家/城市 | China's population（中国的人口）                             |
| 表示拟人化    | nature's beauty（自然之美）                                  |

#### （5）所有格的省略用法

| 情况          | 示例                                                   |
| :------------ | ------------------------------------------------------ |
| 表示商店/住宅 | at the **doctor's**（在诊所/医生家）                   |
| 表示教堂/寺庙 | St. Paul's（圣保罗大教堂）                             |
| 避免重复      | My car is faster than **Tom's**.（我的车比汤姆的快。） |


### 5. 不可数名词的量化表达

不可数名词不能直接用数字修饰，需要借助量词：

- a glass of **water**（一杯水）
- a piece of **information**（一条信息）
- a cup of **tea**（一杯茶）
- a bottle of **milk**（一瓶牛奶）


## 第二部分 · 代词（Pronoun）

> **代词**是用来代替名词或名词短语的词，避免重复，使语言更简洁。


### 1. 人称代词（Personal Pronoun）

#### （1）人称代词一览表

|     人称     |        主格（作主语）         |         宾格（作宾语）         |
| :----------: | :---------------------------: | :----------------------------: |
| 第一人称单数 |            I（我）            |            me（我）            |
| 第二人称单数 |           you（你）           |           you（你）            |
| 第三人称单数 | he（他）/ she（她）/ it（它） | him（他）/ her（她）/ it（它） |
| 第一人称复数 |          we（我们）           |           us（我们）           |
| 第二人称复数 |          you（你们）          |          you（你们）           |
| 第三人称复数 |      they（他/她/它们）       |       them（他/她/它们）       |

> 💡 **注意**：you 既是单数又是复数；it 既可作主格也可作宾格。

#### （2）用法示例

- **I** am a student.（我是一名学生。——主格作主语）
- Please tell **me** the truth.（请告诉我真相。——宾格作宾语）
- **She** likes reading.（她喜欢阅读。——主格作主语）
- Listen to **her**.（听她说。——宾格作介词宾语）
- **We** are friends.（我们是朋友。——主格作主语）
- Can you help **us**?（你能帮我们吗？——宾格作宾语）
- **They** are from China.（他们来自中国。——主格作主语）
- I'll call **them** later.（我待会儿给他们打电话。——宾格作宾语）

#### （3）it 的特殊用法

|     用法     | 示例                                                      |
| :----------: | --------------------------------------------------------- |
|   指代天气   | **It** is raining.（正在下雨。）                          |
|   指代时间   | **It** is 8 o'clock.（现在是8点。）                       |
|   指代距离   | **It** is 5 miles to the station.（到车站有5英里。）      |
|  作形式主语  | **It** is important to learn English.（学英语很重要。）   |
|  作形式宾语  | I find **it** easy to do this.（我发现做这件事很容易。）  |
| 用于强调句型 | **It** is Tom who broke the window.（打破窗户的是汤姆。） |

#### （4）并列人称代词的使用顺序

|      情况       | 顺序             | 示例                                                    |
| :-------------: | ---------------- | ------------------------------------------------------- |
| 肯定句（单数）  | you → he/she → I | You, he and I are all right.（你、他和我都没事。）      |
| 肯定句（复数）  | we → you → they  | We, you and they should go.（我们、你们和他们应该去。） |
| 承认错误/担责时 | I → you → he     | I, you and he are to blame.（我、你和他都有责任。）     |


### 2. 物主代词（Possessive Pronoun）

#### （1）物主代词一览表

|     人称     |        形容词性（+ 名词）         |         名词性（单独使用）         |
| :----------: | :-------------------------------: | :--------------------------------: |
| 第一人称单数 |            my（我的）             |            mine（我的）            |
| 第二人称单数 |           your（你的）            |           yours（你的）            |
| 第三人称单数 | his / her / its（他的/她的/它的） | his / hers / its（他的/她的/它的） |
| 第一人称复数 |           our（我们的）           |           ours（我们的）           |
| 第二人称复数 |          your（你们的）           |          yours（你们的）           |
| 第三人称复数 |   their（他们的/她们的/它们的）   |   theirs（他们的/她们的/它们的）   |

> 💡 **核心区别**：
> - **形容词性** → 后面必须接名词（This is **my** book. 这是我的书。）
> - **名词性** → 后面不接名词，可单独使用（This book is **mine**. 这本书是我的。）

**记忆口诀：有名则形，无名则名。**

#### （2）用法示例

|   类型   | 示例                                                         | 说明                                |
| :------: | ------------------------------------------------------------ | ----------------------------------- |
| 形容词性 | This is **my** book.（这是我的书。）                         | my + 名词 book                      |
|  名词性  | This book is **mine**.（这本书是我的。）                     | mine 单独使用，相当于 my book       |
| 形容词性 | Is this **your** pen?（这是你的钢笔吗？）                    | your + 名词 pen                     |
|  名词性  | The pen is **yours**.（这支钢笔是你的。）                    | yours 单独使用，相当于 your pen     |
| 形容词性 | I like **her** hair.（我喜欢她的头发。）                     | her + 名词 hair                     |
|  名词性  | Her hair is longer than **his**.（她的头发比他的长。）       | his 单独使用，相当于 his hair       |
| 形容词性 | **Their** house is big.（他们的房子很大。）                  | their + 名词 house                  |
|  名词性  | Our house is bigger than **theirs**.（我们的房子比他们的大。） | theirs 单独使用，相当于 their house |


### 3. 反身代词（Reflexive Pronoun）

#### （1）反身代词一览表

|     人称     |                        反身代词                        |
| :----------: | :----------------------------------------------------: |
| 第一人称单数 |                  **myself**（我自己）                  |
| 第二人称单数 |                 **yourself**（你自己）                 |
| 第三人称单数 | **himself / herself / itself**（他自己/她自己/它自己） |
| 第一人称复数 |               **ourselves**（我们自己）                |
| 第二人称复数 |               **yourselves**（你们自己）               |
| 第三人称复数 |      **themselves**（他们自己/她们自己/它们自己）      |

> 💡 **记忆规律**：
> - 第一、二人称：**形容词性物主代词 + self/selves**（myself, yourself, ourselves, yourselves）
> - 第三人称：**宾格代词 + self/selves**（himself, herself, itself, themselves）

#### （2）用法示例

|          用法          | 示例                                                         |
| :--------------------: | ------------------------------------------------------------ |
| 作宾语（动作回到自身） | I taught **myself** English.（我自学了英语。）               |
|                        | She hurt **herself**.（她伤到了自己。）                      |
|                        | He cut **himself** while shaving.（他刮胡子时割伤了自己。）  |
| 作同位语（强调"亲自"） | I **myself** saw it.（我亲眼看到的。）                       |
|                        | The president **himself** visited the school.（校长本人参观了学校。） |
|                        | You should do it **yourself**.（你应该自己做。）             |

**固定搭配：**
- enjoy oneself（玩得开心）
- by oneself（独自地）
- help oneself to（随便吃/用）
- teach oneself（自学）
- devote oneself to（致力于）
- say to oneself（心里想）
- come to oneself（苏醒过来）


### 4. 指示代词（Demonstrative Pronoun）

|       代词        | 用法               |                              示例 |
| :---------------: | :----------------- | --------------------------------: |
| **this**（这个）  | 指代近处的单数事物 | This is my pen.（这是我的钢笔。） |
| **that**（那个）  | 指代远处的单数事物 | That is your bag.（那是你的包。） |
| **these**（这些） | 指代近处的复数事物 | These are apples.（这些是苹果。） |
| **those**（那些） | 指代远处的复数事物 |    Those are books.（那些是书。） |

> 💡 **this / these** 指时间或空间上较近的人或物；**that / those** 指时间或空间上较远的人或物。

**特殊用法：**
- He told me a story. **That** was very interesting.（他给我讲了一个故事。那很有趣。——that 指代上文的 story）
- **This** is what I want to say: you should study hard.（这就是我想说的：你应该努力学习。——this 指代下文的内容）
- Hello, **this** is Tom speaking.（你好，我是汤姆。——电话用语）
- The weather in Beijing is colder than **that** in Shanghai.（北京的天气比上海的冷。——that 代替前面的 weather，避免重复）


### 5. 不定代词（Indefinite Pronoun）详解

#### （1）some / any

|   代词   | 用法                                    | 示例                                                         |
| :------: | --------------------------------------- | ------------------------------------------------------------ |
| **some** | 用于肯定句；表请求/建议的疑问句         | I have **some** money.（我有一些钱。）                       |
|          |                                         | Would you like **some** coffee?（你想来点咖啡吗？）          |
| **any**  | 用于否定句/疑问句；肯定句中表"任何一个" | Do you have **any** questions?（你有问题吗？）               |
|          |                                         | I don't have **any** money.（我没有钱。）                    |
|          |                                         | You can take **any** book you like.（你可以拿你喜欢的任何一本书。） |

#### （2）all / both / neither / none / either

|    代词     |   含义   |      指代范围      | 示例                                           |
| :---------: | :------: | :----------------: | ---------------------------------------------- |
|   **all**   |   全部   |     三者或以上     | All of us are here.（我们所有人都在这里。）    |
|  **both**   |  两者都  |        两者        | Both of them are right.（他们两个都对。）      |
| **neither** | 两者都不 |    两者（否定）    | Neither answer is correct.（两个答案都不对。） |
|  **none**   | 没有一个 | 三者或以上（否定） | None of them came.（他们一个都没来。）         |
| **either**  | 两者之一 |   两者（二选一）   | You can take either one.（你可以拿任意一个。） |

> 💡 **小结**：both/neither/either 用于两者；all/none 用于三者或以上。

#### （3）every / each

|   代词    | 强调角度 |  指代范围  | 用法                                                         |
| :-------: | :------: | :--------: | ------------------------------------------------------------ |
| **each**  | 强调个体 | 两者或以上 | 可单独使用（Each of us has a task. 我们每个人都有一个任务。） |
| **every** | 强调全体 | 三者或以上 | 不可接 of（Every student passed. 每个学生都通过了。）        |

#### （4）many / much / few / little

|     代词     |       含义       |   修饰名词类型   |
| :----------: | :--------------: | :--------------: |
|   **many**   |       许多       | 可数名词（复数） |
|   **much**   |       许多       |    不可数名词    |
|   **few**    | 几乎没有（否定） | 可数名词（复数） |
|  **a few**   |  有几个（肯定）  | 可数名词（复数） |
|  **little**  | 几乎没有（否定） |    不可数名词    |
| **a little** |  有一点（肯定）  |    不可数名词    |

> 💡 **记忆口诀：有 a 是肯定，无 a 是否定；few 接可数，little 接不可数。**

**用法示例：**
- Many people like music.（许多人喜欢音乐。）
- Much water is wasted.（很多水被浪费了。）
- I have few friends.（我几乎没有朋友。——否定）
- I have a few friends.（我有几个朋友。——肯定）
- There is little milk left.（几乎没剩下牛奶了。——否定）
- There is a little milk left.（还剩一点牛奶。——肯定）

#### （5）one / it

|  代词   | 用法                                 | 示例                                                         |
| :-----: | ------------------------------------ | ------------------------------------------------------------ |
| **one** | 替代同类事物中的一个（泛指，非原物） | I lost my pen. I need to buy **one**.（我丢了钢笔，我需要买一支。——不是原来那支） |
| **it**  | 替代同一个事物（特指，即原物）       | I lost my pen. I need to find **it**.（我丢了钢笔，我需要找到它。——就是原来那支） |

#### （6）other / the other / the others / another

|      代词      |     含义      | 指代范围             |
| :------------: | :-----------: | -------------------- |
|   **other**    |    其他的     | 泛指                 |
| **the other**  | 另一个/剩下的 | 两者中的另一个       |
| **the others** |  其余的全部   | 三者以上中的全部剩余 |
|  **another**   |    另一个     | 三者以上中的再一个   |

**用法示例：**
- I have two pens. One is red, **the other** is blue.（我有两支笔。一支是红色的，另一支是蓝色的。——两者中的另一个）
- Some students are reading, **the others** are writing.（一些学生在阅读，其余的在写字。——三者以上中的全部剩余）
- I don't like this one. Please give me **another**.（我不喜欢这个，请再给我一个。——三者以上中的另一个）

> 💡 **记忆口诀：两者中 the other，三者以上 another；泛泛而谈用 others，全部剩余 the others。**


### 6. 连接代词（Conjunctive Pronoun）

引导名词性从句并在从句中充当句子成分。

|     代词      |         含义         | 在从句中的作用   |
| :-----------: | :------------------: | ---------------- |
|    **who**    |          谁          | 作主语           |
|   **whom**    |          谁          | 作宾语           |
|   **whose**   |         谁的         | 作定语           |
|   **what**    |     什么、所……的     | 作主语/宾语/表语 |
|   **which**   | 哪一个（有选择范围） | 作主语/宾语/定语 |
|  **whoever**  |        无论谁        | 作主语           |
| **whatever**  |       无论什么       | 作主语/宾语/表语 |
| **whichever** |      无论哪一个      | 作主语/宾语/定语 |

**用法示例：**
- I know **who** he is.（我知道他是谁。）
- **What** he said is true.（他说的是真的。）
- **Whoever** comes will be welcomed.（无论谁来都会受到欢迎。）
- I will do **whatever** you ask.（我会做你要求的任何事。）


### 7. 疑问代词（Interrogative Pronoun）

引导特殊疑问句，对人或事物进行提问。

|        代词         | 用法                     | 示例                                    |
| :-----------------: | ------------------------ | --------------------------------------- |
|    **who**（谁）    | 提问主语或宾语（指人）   | Who is that?（那是谁？）                |
|   **whom**（谁）    | 提问宾语（指人，更正式） | Whom did you see?（你看见了谁？）       |
|  **whose**（谁的）  | 提问所属关系             | Whose is this pen?（这是谁的钢笔？）    |
|  **what**（什么）   | 提问事物或身份           | What is that?（那是什么？）             |
| **which**（哪一个） | 提问选择范围             | Which is your book?（哪一本是你的书？） |


### 8. 关系代词（Relative Pronoun）

引导定语从句，连接主句和从句。（详细内容见"定语从句"章节）

|   代词    | 用法                   | 示例                                                         |
| :-------: | ---------------------- | ------------------------------------------------------------ |
|  **who**  | 指人，作主语           | The man **who** lives there is a teacher.（住在那里的那个人是老师。） |
| **whom**  | 指人，作宾语           | The girl **whom** I met is his sister.（我遇到的那个女孩是他妹妹。） |
| **which** | 指物，作主语或宾语     | The book **which** I bought is interesting.（我买的那本书很有趣。） |
| **that**  | 指人或物，作主语或宾语 | The movie **that** we watched was great.（我们看的那部电影很棒。） |
| **whose** | 指人或物，作定语       | The boy **whose** father is a doctor is my friend.（父亲是医生的那个男孩是我的朋友。） |


## ⚠️ 常见易错点汇总

|           易错点            |                            ❌ 错误                            |                                                       ✅ 正确 |
| :-------------------------: | :----------------------------------------------------------: | -----------------------------------------------------------: |
|        主格/宾格混淆        |              Me am a student.（我是一名学生。）              |                                          **I** am a student. |
|        主格/宾格混淆        |          Please tell I the truth.（请告诉我真相。）          |                                Please tell **me** the truth. |
| 形容词性/名词性物主代词混淆 |              This book is my.（这本书是我的。）              |                                       This book is **mine**. |
|        反身代词误用         |            I taught me English.（我自学了英语。）            |                                 I taught **myself** English. |
|       some / any 混用       |         Do you have some questions?（你有问题吗？）          |                               Do you have **any** questions? |
|      few / a few 混淆       | I have few friends, so I'm happy.（我几乎没有朋友，所以我很快乐。——前后矛盾） | I have **a few** friends, so I'm happy.（我有几个朋友，所以我很快乐。） |
|      few / a few 混淆       |      I have a few money.（我有一些钱。——money 不可数）       |                                   I have **a little** money. |
|     neither / none 混淆     | Neither of the three students came.（三个学生都没来。——三者用 none） |                         **None** of the three students came. |
|      every / each 误用      | Every of us has a book.（我们每个人都有一本书。——every 不能接 of） |                                   **Each** of us has a book. |
|        one / it 混淆        | I have a dog. One is friendly.（我有一只狗。它很友好。——同一只狗，用 it） |                            I have a dog. **It** is friendly. |
|       other 系列混淆        | I have two pens. One is red, another is blue.（我有两支笔。一支是红的，另一支是蓝的。——两者用 the other） |          I have two pens. One is red, **the other** is blue. |
|       某国人复数写错        |                   two Germen（两个德国人）                   |                                              two **Germans** |
|         所有格错误          |                    Toms book（汤姆的书）                     |                                               **Tom's** book |
|     可数/不可数名词误用     |            some informations（一些信息——不可数）             |                                         some **information** |
|       不规则复数记错        |                    two childs（两个孩子）                    |                                             two **children** |
|       不规则复数记错        |                   three mouses（三只老鼠）                   |                                               three **mice** |
|      单复数同形误加 -s      |                  a lot of sheeps（很多羊）                   |                                           a lot of **sheep** |


## 📌 小结

|        词类         | 核心功能           | 关键考点                                                     |
| :-----------------: | ------------------ | ------------------------------------------------------------ |
|  **名词（Noun）**   | 表示人、事物、概念 | 可数/不可数，规则与不规则复数，集体名词主谓一致，某国人表达，名词所有格 |
| **代词（Pronoun）** | 代替名词           | 人称代词（主格/宾格）、物主代词、反身代词、指示代词、不定代词（some/any, few/little, each/every, one/it, other系列等）、连接代词、疑问代词、关系代词 |
