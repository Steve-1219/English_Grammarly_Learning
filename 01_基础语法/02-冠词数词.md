# 第二章 · 冠词、数词

> 📚 **冠词**是名词的"帽子"，**数词**是数量的"标尺"。掌握好它们，表达更精准。


## 第一部分 · 冠词（Article）

> **冠词**是一种虚词，本身不能单独使用，常放在名词前起限定作用。冠词分为不定冠词 a/an、定冠词 the 和零冠词（不用冠词的情况）。


### 一、不定冠词（Indefinite Article）

不定冠词 a/an 常用于可数名词单数前，表示泛指或表示数量"一"。

#### 1. a / an 的区分

|  冠词  | 使用条件                       | 示例                                                         |
| :----: | ------------------------------ | ------------------------------------------------------------ |
| **a**  | 用于以**辅音音素**开头的单词前 | a book（一本书）, a day（一天）, a table（一张桌子）, a car（一辆车）, a house（一所房子）, a jacket（一件夹克） |
| **an** | 用于以**元音音素**开头的单词前 | an apple（一个苹果）, an umbrella（一把雨伞）, an English book（一本英语书）, an egg（一个鸡蛋）, an orange（一个橙子）, an interesting story（一个有趣的故事） |

> ⚠️ **注意**：是看**读音**（音素）而不是看首字母！
> - **a university**（虽然 u 是元音字母，但读音以 /juː/ 开头，是辅音音素）
> - **an hour**（虽然 h 是辅音字母，但 h 不发音，读音以元音音素开头）
> - **an honest man**（h 不发音，以元音音素开头）
> - **a European**（European 以 /jʊə/ 开头，是辅音音素）

#### 2. 不定冠词的用法

|          用法          | 说明                                   | 示例                                                         |
| :--------------------: | -------------------------------------- | ------------------------------------------------------------ |
|        **泛指**        | 表示一类人或物中的一个                 | I met a beautiful girl yesterday.（昨天我遇见了一个漂亮的女孩。） |
|                        |                                        | He started eating an apple.（他开始吃苹果。）                |
|    **表示数量"一"**    | 但数的概念没有 one 强烈                | The bridge can support a heavy truck.（这座桥能承受一辆重型卡车。） |
|                        |                                        | Rome was not built in a day.（罗马非一日建成。）             |
|      **固定搭配**      | 常用于固定短语中                       | a lot of（许多）, make an effort（努力）, have a good time（玩得开心）, a bit（一点儿）, a little（一点儿）, have a cold（感冒）, take a walk（散步）, in a hurry（急匆匆）, have a rest（休息一会儿） |
| **表示价格/速度/频率** | 相当于"每"、"每一"                     | He goes to the library four times a week.（他每周去图书馆四次。） |
|                        |                                        | I can type 35 words a minute.（我每分钟能打35个单词。）      |
|                        |                                        | He was driving at 30 miles an hour.（他以每小时30英里的速度驾车。） |
|                        |                                        | It costs 50 pence a kilo.（价格是每公斤50便士。）            |
| **表示身份/职业/地位** | 用于单数可数名词前表示职业、身份等     | Lucy is an English teacher.（露西是一名英语老师。）          |
|                        |                                        | You are a student.（你是一名学生。）                         |
|                        |                                        | She's a Buddhist.（她是一个佛教徒。）                        |
|   **用于集体名词前**   | 对整体进行描述                         | Lucy and Mark are a couple.（露西和马克是一对夫妇。）        |
|                        |                                        | His speech attracted a large audience.（他的演讲吸引了一大批观众。） |
|   **用于物质名词前**   | 使之转化为具体名词，表示"一场""一杯"等 | They were caught in a heavy snow.（他们遇到了一场大雪。）    |
|                        |                                        | I made a coffee.（我冲了一杯咖啡。）                         |
|   **用于专有名词前**   | 表示某个不认识的人或具有某种特征的人   | I want to be a Lu Xun.（我想成为鲁迅那样的人。）             |
|                        |                                        | A Mr. White wants to see you.（一位怀特先生想见你。）        |
|   **用于抽象名词前**   | 使抽象名词具体化                       | Yesterday's performance was a great success.（昨天的演出是一个巨大的成功。） |
|                        |                                        | Working with Jane has been a great joy.（和简一起工作是一件令人愉快的事。） |


### 二、定冠词（Definite Article）

定冠词 the 用于名词前，表示**特指**的人或物。

|            用法            | 说明                               | 示例                                                         |
| :------------------------: | ---------------------------------- | ------------------------------------------------------------ |
|          **特指**          | 特指某人或某物                     | The girl in red is my sister.（穿红色衣服的女孩是我的妹妹。） |
|                            |                                    | He took the medicine.（他把药吃了。）                        |
|      **谈话双方共知**      | 双方都明确的人或物                 | Could you please help me open the window?（你可以帮我打开窗户吗？） |
|                            |                                    | Six of the 38 people were U.S. citizens.（38人中有6个是美国公民。） |
|  **复述上文提到过的事物**  | 第一次用 a/an，第二次用 the        | There is a book on the table, and the book is used for learning English.（桌上有一本书，这本书是用来学习英语的。） |
|                            |                                    | They bought a hotel. I have visited the hotel.（他们买了一家酒店。我曾经去过那家酒店。） |
|    **序数词/最高级前**     | 序数词和形容词最高级前必须加 the   | I got the first prize in the competition.（我在比赛中获得了一等奖。） |
|                            |                                    | I live on the second floor.（我住在二楼。）                  |
|                            |                                    | Jack is the tallest student in our class.（杰克是我们班上最高的学生。） |
|                            |                                    | I am the happiest man in the world.（我是世界上最幸福的人。） |
|       **西洋乐器前**       | 西洋乐器名前加 the（民族乐器不加） | Mark can play the piano, but he can't play Erhu.（马克会弹钢琴，但不会拉二胡。） |
|                            |                                    | Do you play the guitar?（你弹吉他吗？）                      |
|  **"形容词/分词 + the"**   | 表示一类人                         | the young（年轻人）, the old（老年人）, the wounded（伤员）, the sick（病人）, the disabled（残疾人） |
|     **"the + 比较级"**     | 表示"越……越……"                     | The harder you study, the higher score you will get.（你学习越努力，获得的分数就越高。） |
|                            |                                    | The more she learns, the more she wants to learn.（她学得越多，就越想学。） |
|       **方位/时间**        | 表示方位、时间的短语               | in the north（在北方）, in the morning（在早上）, in the future（未来）, around the corner（在附近）, at the top of（在……的顶端）, on the left（在左边） |
|       **姓氏复数前**       | 表示该姓氏一家人或夫妇二人         | The Greens love each other deeply.（格林夫妇深爱着彼此。）   |
|                            |                                    | The Blacks are on holiday now.（布莱克一家正在度假。）       |
|     **独一无二的事物**     | 世界上唯一的事物                   | the sun（太阳）, the earth（地球）, the moon（月球）         |
| **普通名词构成的专有名词** | 由普通名词构成的专有名词前         | the Great Wall（长城）, the Silk Road（丝绸之路）, the University of Cambridge（剑桥大学） |
|       **地理名词前**       | 江河、湖泊、海洋、山脉、群岛等     | the Yellow River（黄河）, the Pacific Ocean（太平洋）, the English Channel（英吉利海峡） |
|     **党派/团体/机构**     | 党派、团体、机构等名词前           | the Communist Party（共产党）, the Democratic Party（民主党）, the United Nations（联合国） |

> 💡 **定冠词记忆口诀**：
> 特指双熟悉，上文已提及，世上独一无二，序数最高级，某些专有名，习语及乐器，党派及地理。


### 三、零冠词（Zero Article）

零冠词是指名词前**不使用冠词**的情况。

|             用法              | 说明                                                 | 示例                                                         |
| :---------------------------: | ---------------------------------------------------- | ------------------------------------------------------------ |
| **三餐/球类/棋类/学科/语言**  | 表示一日三餐、球类、棋类、学科、语言的名词前不加冠词 | have breakfast（吃早饭）, play chess（下象棋）, learn English（学英语）, speak Chinese（讲汉语）, play basketball（打篮球） |
|    **季节/月份/星期/节日**    | 表示季节、月份、星期、节日的名词前不加冠词           | in summer（在夏天）, on Children's Day（在儿童节）, at Christmas（在圣诞节）, in July（在七月）, on Monday（在星期一） |
|       **by + 交通工具**       | by + 交通工具名词前不加冠词                          | by bus（乘公交）, by train（乘火车）, by airplane（乘飞机）  |
|   **名词前已有其他限定词**    | 名词前已有物主代词、指示代词、所有格等，不加冠词     | On his birthday she sent him presents.（他生日那天，她送了他礼物。） |
|                               |                                                      | This is Lily's car.（这是莉莉的车。）                        |
|                               |                                                      | I'm checking this bag.（我正在检查这个包。）                 |
|         **固定搭配**          | 某些固定搭配中不加冠词                               | lose heart（灰心）, on foot（步行）, on fire（着火）         |
|      **物质名词表泛指**       | 物质名词表示泛指时不加冠词                           | Water boils at 100℃.（水在100摄氏度沸腾。）                  |
|                               |                                                      | Plastic is hard to break down.（塑料很难分解。）             |
|      **抽象名词表泛指**       | 抽象名词表示泛指时不加冠词                           | Work is a necessary thing in life.（工作是人生中必不可少的一件事。） |
|                               |                                                      | Unity is strength.（团结就是力量。）                         |
|                               |                                                      | Failure is the mother of success.（失败是成功之母。）        |
| **人名/城市/街道/大多数国名** | 专有名词前不加冠词                                   | Beijing is the capital of China.（北京是中国的首都。）       |
|                               |                                                      | Joan lives in Bridge Street.（琼住在大桥街。）               |
|      **复数名词表泛指**       | 复数名词表示一类人或事物时不加冠词                   | Students should study hard.（学生应该努力学习。）            |
|                               |                                                      | Horses need to be exercised regularly.（马需要经常锻炼。）   |
|    **独一无二的职位/头衔**    | 表示独一无二的职位、头衔的名词前不加冠词             | William was elected president of the company.（威廉当选为公司总裁。） |
|                               |                                                      | Lincoln, president of the United States, was murdered in 1865.（美国总统林肯于1865年被谋杀。） |

#### 交通工具前加冠词的补充说明

> 💡 表示交通工具的名词不接在介词 by 后时，则名词前要使用冠词、物主代词、指示代词等限定。
> - I take **a bus** to work every day.（我每天乘公交车上班。）
> - I've left my bag on **the bus**.（我把包丢在公交车上了。）
> - She often goes to school on **her bike**.（她经常骑自行车上学。）

#### 加 the 与不加 the 含义对比

|   不加 the   |  含义  |      加 the      |   含义   |
| :----------: | :----: | :--------------: | :------: |
|  take place  |  发生  |  take the place  |   代替   |
|   on earth   |  究竟  |   on the earth   | 在地球上 |
|    by sea    |  乘船  |    by the sea    |  在海边  |
| in charge of | 负责…… | in the charge of | 由……负责 |
|   in class   | 在上课 |   in the class   |  在班上  |

> 💡 **零冠词记忆口诀**：
> 下列情况应免冠，代词限定名词前；
> 专有名词不可数，学科球类三餐饭；
> 复数名词表泛指，节日星期月份前；
> 颜色语种和国名，称呼习语及头衔。


## 第二部分 · 数词（Numeral）

> **数词**是表示数目多少和顺序多少的词。数词分为基数词和序数词，基数词表示数目，序数词表示次序。


### 一、基数词（Cardinal Numbers）

#### 1. 基数词的写法

|  1-10   |    11-19     | 20-90（整十） |         100及以上         |
| :-----: | :----------: | :-----------: | :-----------------------: |
|  1 one  |  11 eleven   |   20 twenty   |      100 one hundred      |
|  2 two  |  12 twelve   |   30 thirty   |    1,000 one thousand     |
| 3 three | 13 thirteen  |   40 forty    |   1,000,000 one million   |
| 4 four  | 14 fourteen  |   50 fifty    | 1,000,000,000 one billion |
| 5 five  |  15 fifteen  |   60 sixty    |                           |
|  6 six  |  16 sixteen  |  70 seventy   |                           |
| 7 seven | 17 seventeen |   80 eighty   |                           |
| 8 eight | 18 eighteen  |   90 ninety   |                           |
| 9 nine  | 19 nineteen  |               |                           |
| 10 ten  |              |               |                           |

> 💡 **补充提示**：
> - 13-19：由 3-9 加后缀 **-teen** 构成，注意 **eighteen** 只保留一个 t，**thirteen**、**fifteen** 分别由 three、five 转化而来。
> - 20-90：由 2-9 加后缀 **-ty** 构成，注意 **eight** 只保留一个 t，**twenty**、**thirty**、**forty**、**fifty** 分别由 two、three、four、five 转化而来。
> - 21-99：整十数 + 个位数，中间加连字符 **"-"**，如 **twenty-four**。

#### 2. 基数词的基本用法

基数词用于描述事物数量的多少，在句中常作主语、宾语、表语、定语等。

- **Two** of the girls are from Wuhan.（女孩子中有两个来自武汉。——作主语）
- There are **four** students in our group.（我们组里有四名学生。——作定语）
- Over **one thousand** people died during the war.（有一千多人在这场战争中丧命。——作定语）
- The city has a population of **two million**.（这个城市有两百万人口。——作宾语）
- Two and two is **four**.（二加二等于四。——作表语）


### 二、序数词（Ordinal Numbers）

#### 1. 序数词的写法

|    1-10     |      11-19       |  20-90（整十）  |
| :---------: | :--------------: | :-------------: |
|  1st first  |  11th eleventh   | 20th twentieth  |
| 2nd second  |   12th twelfth   | 30th thirtieth  |
|  3rd third  | 13th thirteenth  |  40th fortieth  |
| 4th fourth  | 14th fourteenth  |  50th fiftieth  |
|  5th fifth  |  15th fifteenth  |  60th sixtieth  |
|  6th sixth  |  16th sixteenth  | 70th seventieth |
| 7th seventh | 17th seventeenth | 80th eightieth  |
| 8th eighth  | 18th eighteenth  | 90th ninetieth  |
|  9th ninth  | 19th nineteenth  |                 |
| 10th tenth  |                  |                 |

> 💡 **补充提示**：
> - 除 first, second, third 外，序数词通常由基数词加后缀 **-th** 构成，注意 **fifth**、**eighth**、**ninth**、**twelfth** 的拼写形式。
> - 缩写形式：阿拉伯数字 + 序数词最后两个字母，如 **1st, 2nd, 3rd, 4th, 21st** 等。
> - 两位数及以上：只需把个位数变序数词，十位数不变，如 **twenty-third**、**one hundred and fifty-sixth**。

#### 2. 序数词的基本用法

序数词用于描述事物的排列顺序，在句中常作主语、表语、定语等。

- I am always the **first** to come to the classroom.（我总是第一个来教室的人。——作表语）
- I live on the **third** floor.（我住在三楼。——作定语）
- The **eighth** of this month is my birthday.（这个月的8号是我的生日。——作主语）
- Mary was the **second** to arrive.（玛丽是第二个到的。——作表语）


### 三、数词的其他用法

#### 1. 表确切和不确切数目

|      类型      | 规则                                                         | 示例                                                         |
| :------------: | ------------------------------------------------------------ | ------------------------------------------------------------ |
|  **确切数目**  | hundred, thousand, million, billion, dozen, score 等前有具体数字时，不加 -s | two hundred students（两百名学生）, six billion dollars（六十亿美元）, two dozen eggs（两打鸡蛋） |
| **不确切数目** | 上述词 + -s + of，表示"数百""数千"等                         | hundreds of teachers（数百名老师）, dozens of eggs（几十个鸡蛋）, thousands of people（数千人）, scores of young doctors（许多年轻的医生） |

> 💡 当这些词前有 some, several, a few, many 等词限定时，用单数或复数均可：
> - many thousand (of) books = many thousands of books（几千本书）

#### 2. 表分数和百分数

|    类型    | 构成规则                                                     | 示例                                                         |
| :--------: | ------------------------------------------------------------ | ------------------------------------------------------------ |
|  **分数**  | 分子用基数词，分母用序数词。分子为 1 时分母用单数，分子大于 1 时分母用复数 | one third（三分之一）, two fifths（五分之二）, seven tenths（十分之七） |
| **百分数** | 基数词 + percent + of + 名词                                 | forty percent of the students（百分之四十的学生）            |

#### 3. 表倍数

|    倍数    | 表达方式           |
| :--------: | ------------------ |
|    两倍    | **twice / double** |
| 三倍及以上 | **基数词 + times** |

- seven times（七倍）, fifteen times（十五倍）, thirty times（三十倍）

> 💡 倍数常见句型详见核心语法第五章·状语从句。

#### 4. 表年龄

|     类型     | 表达方式                                    | 示例                                                         |
| :----------: | ------------------------------------------- | ------------------------------------------------------------ |
| **确切年龄** | 基数词 + years old / at the age of + 基数词 | He is ten (years old) this year.（他今年十岁。）             |
|              |                                             | At the age of twenty-eight, I got married.（二十八岁时我结婚了。） |
| **大概年龄** | in + one's + 整十复数基数词                 | in my twenties（在我二十多岁时）                             |

> 💡 **注意**：表示"一个十岁的男孩"只能用单数形式：**a ten-year-old boy**

#### 5. 表时间

**（1）表日期：** 月 + 日 + 年 或 日 + 月 + 年
- **October 1(st), 2010** = **1(st) October, 2010**（2010年10月1日）

**月份缩写：**

|   月份   |  缩写  |   月份    | 缩写 |
| :------: | :----: | :-------: | :--: |
| January  |  Jan.  |   July    | Jul. |
| February |  Feb.  |  August   | Aug. |
|  March   |  Mar.  | September | Sep. |
|  April   |  Apr.  |  October  | Oct. |
|   May    | 无缩写 | November  | Nov. |
|   June   |  Jun.  | December  | Dec. |

**（2）表世纪和年代：**
- **the 1980s** / **the 1980's**（20世纪80年代）
- **the eighteenth century** / **the 18th century**（18世纪）
- **the 1900s** / **the 1900's**（20世纪）

**（3）表时刻：**

|        表达方式         | 规则                          | 示例                                                         |
| :---------------------: | ----------------------------- | ------------------------------------------------------------ |
|        **整点**         | 基数词 + o'clock（可省略）    | ten (o'clock)（10:00）                                       |
| **几点几分（≤30分钟）** | 分钟数 + past + 钟点数        | two past nine（9:02）, half past two（2:30）, a quarter past six（6:15） |
| **几点几分（>30分钟）** | (60-分钟数) + to + (钟点数+1) | five to ten（9:55）, a quarter to six（5:45）                |
|      **直接读法**       | 钟点数 + 分钟数               | nine two（9:02）, nine fifty-five（9:55）                    |

**（4）a.m. 和 p.m.**

- **a.m.**（上午），**p.m.**（下午），其前不加 o'clock，且不与 morning / afternoon 同时使用
- **8:10 a.m.** = 8:10 in the morning（上午8点10分）
- **6:00 p.m.** = 6:00 in the afternoon（下午6点）

#### 6. 表编号

| 结构                           | 示例                                                         |
| :----------------------------- | ------------------------------------------------------------ |
| **the + 序数词 + 名词**        | the seventh chapter（第7章）, the fifth lesson（第5课）, the third act（第3幕） |
| **名词（首字母大写）+ 基数词** | Room 403（403房间）, Number 8（第8号）, Chapter 7（第7章）, Lesson 5（第5课）, Act 3（第3幕） |

#### 7. 表加减乘除

|   运算   | 表达方式                                   | 示例                                  |
| :------: | ------------------------------------------ | ------------------------------------- |
| **加法** | 基数词 + and/plus + 基数词 + is + 基数词   | One and/plus two is three.（1+2=3）   |
| **减法** | 基数词 + minus + 基数词 + is + 基数词      | Eight minus three is five.（8-3=5）   |
| **乘法** | 基数词 + times + 基数词 + is + 基数词      | Two times three is six.（2×3=6）      |
| **除法** | 基数词 + divided by + 基数词 + is + 基数词 | Ten divided by two is five.（10÷2=5） |

#### 8. 表小数

| 规则                                               | 示例                              |
| :------------------------------------------------- | --------------------------------- |
| 小数点读作 **point**，小数点后的数字按顺序依次读出 | zero point four（0.4）            |
|                                                    | two point six eight four（2.684） |


## ⚠️ 常见易错点汇总

|            易错点             |                  ❌ 错误                   |                          ✅ 正确 |
| :---------------------------: | :---------------------------------------: | ------------------------------: |
| a / an 混淆（看音素不看字母） |                  a hour                   |                     **an** hour |
| a / an 混淆（看音素不看字母） |               an university               |                **a** university |
|      球类运动前误加 the       |            play the basketball            |             play **basketball** |
|      西洋乐器前漏加 the       |                play piano                 |              play **the** piano |
|        三餐前误加 the         |            have the breakfast             |              have **breakfast** |
|       序数词前漏加 the        |                first prize                |             **the** first prize |
|    复数名词表泛指误加 the     | The students should study hard.（泛指时） | **Students** should study hard. |
|   hundred 表确切数目时加 -s   |           two hundreds students           |        **two hundred** students |
| hundred 表不确切数目时漏加 -s |            hundred of students            |        **hundreds of** students |
|         分数表达错误          |                 two third                 |                  **two thirds** |
|         年龄表达错误          |            a ten-years-old boy            |          a **ten-year-old** boy |


## 📌 小结

|        词类         | 核心功能       | 关键考点                                                     |
| :-----------------: | -------------- | ------------------------------------------------------------ |
| **冠词（Article）** | 限定名词       | 不定冠词 a/an 的区分，定冠词 the 的特指用法，零冠词的多种情况 |
| **数词（Numeral）** | 表示数量和顺序 | 基数词的写法，序数词的写法及变化规则，分数/百分数/倍数/时间/编号等特殊用法 |
